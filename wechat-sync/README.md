# 微信公众号同步操作指南（方案一：Mdnice）

## 概述

本指南帮助你将博客文章同步发布到微信公众号「与卓越运营同行」。
核心工具是 **[Mdnice](https://www.mdnice.com/)**（Markdown Nice），一个专为微信公众号排版设计的在线编辑器。

---

## 一、首次配置（约 10 分钟，只需做一次）

### 步骤 1：注册 Mdnice 账号

1. 打开 https://www.mdnice.com/
2. 点击右上角「登录」，支持 GitHub / 微信扫码登录
3. 登录后进入主界面

### 步骤 2：导入自定义主题

1. 点击左上角「主题」按钮
2. 选择「新建主题」或「导入主题」
3. 主题名称填写：`AI制造卓越运营`
4. 打开本目录下的 `mdnice-theme.css` 文件
5. 复制全部内容，粘贴到 Mdnice 的 CSS 编辑区
6. 点击「保存」

> 这个主题样式与博客网站风格统一：蓝色主色调、圆角卡片、清晰代码高亮。

### 步骤 3：验证主题效果

1. 在 Mdnice 编辑区随便粘贴一段 Markdown 文本
2. 在右侧预览区确认样式效果
3. 检查标题、代码块、表格、引用块是否正常显示

---

## 二、日常同步流程（每篇文章约 5-10 分钟）

### 步骤 1：准备文章内容

在博客目录中选择要同步的文章：

```bash
# 文章位于
blog-app/content/articles/文章名.md
```

打开文章文件，**复制 frontmatter 之后的所有内容**（即 `---` 下方的正文部分，不包含 frontmatter）。

**示例：**
```markdown
---
title: "文章标题"
...（这部分不要复制）
---

# 从这里开始复制 ↓

文章正文内容...

> 本文首发于微信公众号「与卓越运营同行」
> [点击阅读微信公众号原文](...)
```

### 步骤 2：粘贴到 Mdnice

1. 打开 https://www.mdnice.com/
2. 确认主题选择为「AI制造卓越运营」
3. 将复制的内容粘贴到左侧编辑区
4. 右侧预览区会实时显示排版效果

### 步骤 3：检查与调整

检查以下内容是否正常：

- [ ] 标题层级正确（h1 只有一个，作为文章标题）
- [ ] 代码块有语法高亮
- [ ] 表格正常显示（没有错位）
- [ ] 图片链接有效（见下方图片处理说明）
- [ ] 引用块样式正常
- [ ] 数学公式正常渲染（如有）

### 步骤 4：复制到公众号编辑器

1. 点击 Mdnice 右上角的「复制」按钮（会提示「已复制到剪贴板」）
2. 打开微信公众号后台：https://mp.weixin.qq.com/
3. 点击「新建图文」
4. 将光标放在编辑区，**Ctrl+V / Cmd+V 粘贴**
5. 样式会自动带入

### 步骤 5：完善公众号文章信息

在公众号编辑器中完成以下设置：

1. **填写标题**：复制文章标题（frontmatter 中的 title）
2. **作者**：填写「与卓越运营同行」或你的名字
3. **摘要**：复制 frontmatter 中的 summary
4. **封面图**：
   - 上传一张 900×383 像素的封面图
   - 可以从 https://unsplash.com/ 找免费图片
   - 或用 Canva 制作带标题的封面图
5. **原文链接**（可选）：
   - 如果博客已部署上线，可填写博客文章的 URL
   - 读者点击「阅读原文」可跳转到博客

### 步骤 6：预览并发布

1. 点击「预览」，扫码在手机上查看效果
2. 检查手机端排版是否正常
3. 确认无误后，点击「发送」

## 三、图片处理说明

### 问题：博客中的图片链接

博客文章中可能引用了外部图片（如 Unsplash 封面图）。微信公众号要求图片必须上传到微信素材库，外部图片链接在公众号中**无法显示**。

### 解决方案

**方案 A：手动上传图片（推荐）**

1. 在 Mdnice 中，找到文章里的图片
2. 将图片下载到本地
3. 在公众号编辑器中，删除原图片位置
4. 重新上传图片（微信会自动生成素材库链接）

**方案 B：使用 Mdnice 的图床功能**

1. 在 Mdnice 设置中配置图床（支持 GitHub、阿里云 OSS 等）
2. 上传图片后获得固定链接
3. 替换文章中的图片链接

**方案 C：文章不含图片**

如果文章是纯文字 + 代码（如大部分公众号文章），则无需处理图片，直接粘贴即可。

## 四、代码块处理说明

### Mdnice 对代码块的支持

Mdnice 支持完整的代码高亮，语法如下：

````markdown
```python
def hello():
    print("Hello, World!")
```
````

### 公众号中的代码显示效果

- 深色背景 + 浅色文字（与博客风格一致）
- 语法高亮（关键字、字符串、注释等不同颜色）
- 等宽字体
- 自动换行

### 注意事项

- 代码块如果太长（超过 30 行），建议拆分为多个小块
- 公众号不支持横向滚动，长行代码会自动折行
- 如果代码中有 `<` 或 `>` 符号，Mdnice 会自动处理转义

## 六、常见问题

### Q1：粘贴到公众号后样式丢失？

**A**：这通常是因为公众号编辑器对某些 CSS 属性有限制。Mdnice 已做了兼容处理，但如果仍有问题：
- 尝试使用 Chrome 浏览器
- 清除浏览器缓存后重新粘贴
- 检查是否选对了主题

### Q2：表格在手机端显示不全？

**A**：公众号在手机端不支持横向滚动。建议：
- 减少表格列数（最多 4-5 列）
- 缩短单元格内容
- 复杂表格改为列表形式

### Q3：代码块在手机端字体太小？

**A**：Mdnice 默认代码字体 14px，在手机端可能偏小。可以在主题 CSS 中调整：
```css
.Article pre code {
  font-size: 15px; /* 从 14px 改为 15px */
}
```

### Q4：如何修改已发布的公众号文章？

**A**：微信公众号文章发布后**无法修改正文内容**，只能修改标题和摘要。如需修改：
- 删除原文，重新发布（会丢失阅读量和点赞）
- 或在文章下方「留言区」补充更正

### Q5：如何在公众号文章中添加博客原文链接？

**A**：在公众号编辑器中：
1. 选中要添加链接的文字
2. 点击工具栏的「超链接」按钮
3. 选择「外部链接」
4. 填写博客文章的 URL
5. 或在公众号设置中开启「原文链接」，填写博客 URL

> 注意：个人订阅号不支持在正文中添加外部超链接，只能在「阅读原文」位置添加一个链接。如需在正文中加链接，需要企业服务号。
